## Switcher

A beautiful checkbox

### Usage

```
<switcher v-model="checked" />
```

### Props

| name       | type    |Default | Description |
| ---------- | ------- |------- | ----------- |
| disabled | `boolean?` | `false` |             |
| checked  | `boolean?`    | `false` |             |


### Event

| name       | type    | Description |
| ---------- | ------- |------- | ----------- |
| change | `boolean` | - |
